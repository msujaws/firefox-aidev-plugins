# Firefox Desktop Theme Plugin

Skills and tools for Firefox desktop theme reviewers.

## Skills

### `desktop-theme-review`

Review guidance for Firefox desktop theme CSS, design tokens, and lightweight theme integration under `browser/themes/` and `toolkit/themes/`.

Use this skill when reviewing changes to chrome CSS, design tokens, theme plumbing, or SVG icons. See [`skills/desktop-theme-review/SKILL.md`](skills/desktop-theme-review/SKILL.md) for details.
